# Filigran 2026 Report · Chart Sketches

Data visualization sketches for the **State of Threat Management** report — a Filigran whitepaper based on a third-party Vanson Bourne survey of 550 global cybersecurity leaders (Feb–Mar 2026).

**Live site:** https://finalfinalff.github.io/filigran-2026-report-charts/

Twenty-one standalone HTML pages: three interpretations of each of the five whitepaper figures, plus two synthesis "bonus" figures. Each page shows the chart, the data behind it, the relevant Vanson Bourne tabs, and where the chart maps in the whitepaper.

## How to read this

- **Index** — `ff-pages/index.html` lists every chart grouped by whitepaper figure
- **Glossary** — `ff-pages/glossary.html` defines every acronym, cohort, and methodology term used across the charts
- **Each chart page** — chart on top, then "where this maps in the whitepaper" badge, then the underlying data table with source-tab citations

## Source data

- **Survey workbook:** [Filigran 2026 Final Data Report v1.1](https://docs.google.com/spreadsheets/d/1xivLXtpR4c8NOiAGMW1dehRDXJrBUkyG/edit) — Vanson Bourne project `2025112501DG`
